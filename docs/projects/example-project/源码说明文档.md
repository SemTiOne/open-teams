# 示例项目源码说明文档

## 1. 项目定位

- 项目名称：`example-project`
- 项目类型：后端服务
- 在整体系统中的职责：提供示例业务能力，演示模板文档如何落地

## 2. 目录结构

- `src/main`：应用入口
- `src/modules`：业务模块
- `src/shared`：公共能力
- `config/`：配置文件

## 3. 核心运行链

- 启动入口：`src/main`
- 请求入口：`src/modules/*/controller`
- 核心处理链：`controller -> service -> repository`

## 4. 关键边界

- 保持现有模块分层
- 不在控制器层堆叠复杂业务逻辑
- 配置、领域逻辑和数据访问应分离
