# 任务完成检查清单模板

## 完成前确认

- [ ] 需求目标已实现或问题已修复
- [ ] 改动范围与任务目标一致
- [ ] 关键验证已执行
- [ ] 风险与遗留项已明确说明
- [ ] 用户已有改动与本次改动已区分

## 用户确认

- [ ] 已向用户说明完成结论
- [ ] 已说明完成内容与关键路径
- [ ] 已说明验证结果
- [ ] 已说明未完成项或风险项
- [ ] 已请求用户明确确认

## 复盘判断

- [ ] 已判断是否需要复盘
- [ ] 若免复盘，已说明原因
- [ ] 若需要复盘，已输出复盘并等待用户确认

## 资料同步

- [ ] 已判断是否需要同步 `docs/`
- [ ] 已判断是否需要同步 `skills/`
- [ ] 已判断是否需要同步 `references/`
- [ ] 已判断是否需要同步 `change-history/`
