# Open Teams — VS Code Extension

A minimal VS Code extension scaffold for the open-teams project.

## Features

- Adds an `Open Teams: Start` command to the VS Code Command Palette (`Ctrl+Shift+P`)

## Development

```bash
cd extensions/vscode
npm install
npm run compile
```

Then press `F5` in VS Code to launch the Extension Development Host and test the command.

## Usage

1. Open Command Palette (`Ctrl+Shift+P`)
2. Type `Open Teams`
3. Select `Open Teams: Start`
