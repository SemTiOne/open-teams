# 示例

以下示例说明如何在实际对话中触发项目 Scoped Skill：

**示例 1 — bugfix 场景：**

> "看这个项目的 `bugfix` skill，先帮我确认问题范围，再给修复方案。"

**示例 2 — new-feature-dev 场景：**

> "按这个项目的 `new-feature-dev` skill，给我拆一个可验收的实施计划。"

**示例 3 — 文档更新：**

> "用 `docs-update` skill，把这次 API 变更同步更新到 docs/ 和 CHANGELOG。"
