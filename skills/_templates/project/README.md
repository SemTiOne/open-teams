# 项目 Skill 模板

复制本目录后，将 `project` 替换为真实项目名，并在 `scene-template/` 下按场景复制模板。

## 快速使用

```bash
cp -r skills/_templates/project skills/my-project
cd skills/my-project
# 将 scene-template/ 复制为具体场景（如 bugfix、feature-dev）
cp -r scene-template bugfix
```

## 场景定制要点

每个场景的 `SKILL.md` 需要定制：触发条件、前置输入、工作流步骤、产出格式。

详见 `scene-template/SKILL.md` 中的模板说明。
